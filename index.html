    • Eps-nash-algos

    Project maintained by bimatrix-games Hosted on GitHub Pages — Theme by mattgraham

    A collection of algorithms for approximation of Nash equilibrium. The algorithms implemented are stated below with their theoretic upper bounds:

    • DMP - 0.5-NE
    • BBM1 - 0.3819-NE
    • BBM2 - 0.3639-NE
    • TS - 0.3393-NE
    • KS - 0.6667-WSNE
    • KS+ - 0.66076-WSNE
